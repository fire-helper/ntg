The Open Source Definition

  • Source code must be accessible
  • Code must be redistributable
  • Allowance for derived works
  • No discrimation against who gets to use software

The solution: licenses

Permissive

  • Can relicense
  • Modifications can be different license
  • Minimal obligations

MIT, BSD, ISC, etc.

Copyleft

  • Cannot relicense
  • Modifications must be same license
  • More obligations (must release source code, etc.)

GPL, AGPL, MPL, etc.
